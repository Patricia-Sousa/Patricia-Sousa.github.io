UFCD 774

Rede Local - Instalação de Software Base

Formador: Fernando Tavares

Objetivos

Proceder à instalação do sistema de gestão de um servidor de rede local.

Instalar e configurar software cliente.

Caracterizar o sistema operativo de rede.

Reflexão

Uma rede local (LAN) é um grupo de computadores e dispositivos periféricos que partilham uma linha de comunicação comum ou um link sem fios com um servidor, numa área geográfica específica. Uma rede local pode servir apenas dois ou três utilizadores num escritório doméstico ou milhares de utilizadores numa organização. Uma rede local é configurada para que os nós da rede possam comunicar entre si e partilhar recursos, como impressoras ou ficheiros. Um LAN permite que os utilizadores se conectem a servidores internos, sites e outras redes locais que pertencem à mesma rede de longa distância (WAN). As redes de computadores passaram por um longo processo de evolução ao longo dos anos para chegar aos padrões atuais. Como componentes de uma rede, temos os sistemas computacionais, os adaptadores de rede, os sistemas de cablagem e o software de rede. Um sistema operativo de rede é um recurso projetado para conectar vários computadores e partilhar recursos de rede.

Para além das observações anteriores, esta UFCD foi composta pelo desenvolvimento de determinados conteúdos. Aqueles que considero mais importantes são:

- O processo de arranque de um sistema operativo, chamado de boot, no contexto desta UFCD, o processo de arranque do Windows;

- A migração de TI, que pode ser composta num único projeto, por uma migração de dados, de aplicações, de sistema operativo e para a nuvem. (Em aula, foi migrado o Active Directory do Windows Server 2016 para o Windows Server 2022);

- A estrutura do Active Directory;

- Como adicionar um computador Windows 10 a um domínio. (Documentámos como adicionar um computador Ubuntu a um domínio AD DS);

- O processo de instalação do Windows Server 2022, instalação do AD DS, promoção de um controlador de domínio, instalação do DNS e do DHCP;

- O serviço de DNS.

Muitos processos que são realizados através do sistema gráfico de navegação por menus do sistema operativo, podem ser feitos por linha de comando. A linha de comando está presente no Windows desde 1981 e continua a ser útil para executar comandos básicos. O Powershell foi apresentado em 2006 e é considerado a evolução da tradicional linha de comando. Ele permite usar todos os comandos já existentes no CMD, mas é muito mais poderoso, pois é um ambiente de script que permite executar scripts complexos para executar tarefas administrativas do sistema Windows de maneira fácil e eficiente, entre outras funcionalidades. Ao longo das aulas, utilizámos alguns comandos na Shell de comando e também no Powershell, relacionados com os temas em análise.

O maior desafio que senti, ao longo desta UFCD, foi o trabalho autónomo que tive que desenvolver. Foram pedidos vários trabalhos de pesquisa e a sua respetiva documentação, diretamente relacionados com todas as componentes lecionadas durante a UFCD.
