UFCD 772

Sistemas Operativos - Instalação e Configuração

Formador: Fernando Tavares

Objetivos

Instalar e configurar sistemas operativos.

Instalar e configurar utilitários sobre sistemas operativos.

Reflexão

Um sistema operativo é um conjunto de programas que controlam a execução de aplicações e que atua como intermediário entre o utilizador e o hardware. Ele é um software que gere o hardware do computador e fornece um ambiente para a execução de aplicações. Os principais objetivos de um sistema operativo são:

- Tornar o ambiente do computador conveniente e fácil de usar;

- Utilizar o hardware do computador de forma eficiente;

- Executar programas escolhidos pelo utilizador e facilitar a solução de problemas do mesmo.

Existem vários sistemas operativos e entre os mais usados em computadores domésticos encontram-se o Windows, o Linux e o MacOS.

Para ficarmos a conhecer diferentes tipos de sistemas operativos utilizámos o processo de virtualização. A virtualização é um software de aplicação que cria uma camada virtual do hardware sobre o qual corre outro sistema operativo, por exemplo.

É possível instalar vários sistemas operativos numa máquina virtual e para isso instalámos nos nossos computadores o programa Oracle VM VirtualBox. Uma máquina virtual é baseada na arquitetura do computador e proporciona as funcionalidades de um computador físico. Ela é a ferramenta necessária para correr um outro sistema operativo sobre o sistema operativo nativo.

Depois desta instalação, criámos então uma nova máquina virtual com o sistema operativo – Windows 10 Enterprise. A função deste sistema operativo Windows é facilitar o acesso do utilizador ao computador, com interfaces mais eficientes e fáceis de usar, além de integrar softwares mais rápidos e eficazes. Foi feita, através deste processo, uma instalação limpa do sistema operativo. No decorrer da instalação e da sua configuração, criámos uma conta de utilizador padrão. Fomos vendo ao longo das aulas, que existem três contas de utilizador no Windows 10: administrador, utilizador padrão e convidado e percebemos a diferença e as funcionalidades de cada uma.

Nas configurações do Windows 10, explorámos as aplicações: Definições e Painel de controlo, entre outras. Foram abordadas também as ferramentas de manutenção como: desfragmentar e otimizar unidades, limpeza de disco, programador de tarefas e a funcionalidade de segurança e manutenção.

Dentro dos utilitários do Windows 10, considero o Windows Firewall e o Windows Defender, dois programas de muita importância, pois a segurança nos computadores é crucial nos dias de hoje, visto que estamos sujeitos a muitas ameaças digitais.

Vimos também que o Windows 10 tem um novo navegador padrão de internet, o Microsoft Edge. Uma grande vantagem deste navegador é o controlo da privacidade em três níveis, o qual oferece uma boa noção para o utilizador da informação, como está a ser seguida a sua navegação. Entre diversas funcionalidades podemos inclusive observar que tem um enorme potencial de utilização.

Por último, criámos outra máquina virtual para o sistema operativo Ubuntu que é uma distribuição Linux. Este sistema operativo é de código aberto, com a possibilidade de consulta, análise e modificação do código, sem a necessidade de pagar uma licença comercial. No entanto, ainda encontra algumas desvantagens nomeadamente a incompatibilidade de softwares e a incompatibilidade de hardware.

É muito difícil imaginar o funcionamento de um computador sem a existência de um sistema operativo. Ele é extremamente importante para que o hardware (a máquina) funcione de modo eficaz e seguro. O sistema operativo surgiu para tornar a interação do utilizador com o computador mais simples. Sem um sistema operativo num computador, não conseguiríamos por exemplo enviar um email, ouvir música, navegar na internet ou escrever um documento.
