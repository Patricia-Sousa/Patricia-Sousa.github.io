UFCD 799

Sistemas de Rede Local

Formador: António Fonseca

Objetivos

Estruturar e implementar um sistema de rede local.

Implementar políticas de segurança em redes locais.

Reflexão

Para se construir uma rede local (LAN) são necessários pelo menos 2 computadores e o meio de transmissão de dados entre eles. Com os computadores de hoje, qualquer computador na rede pode fornecer serviços ou solicitar serviços, dependendo de como a rede está configurada. Um servidor é um computador destinado a ser um provedor de serviços dedicado e um cliente é um computador que solicita serviços. Uma rede composta por servidores e clientes dedicados é conhecida como rede cliente/servidor. Nesta medida, um sistema operativo cliente é desenhado para desktops ou dispositivos portáteis, enquanto que um sistema operativo de rede é um sistema de computação centralizada que fornece serviços a uma rede de computadores.

As características dos sistemas operativos de rede estão normalmente associadas à administração dos utilizadores, à manutenção do sistema e à funcionalidade de gestão de recursos. Estas incluem o suporte básico, incluindo o suporte de protocolos e processadores, deteção de hardware e multiprocessamento, partilha de bases de dados e sistemas de ficheiros comuns, capacidades de segurança de rede, tais como a autenticação de utilizadores e controlo de acesso, serviços de diretório, backup e serviços web.

Para instalar um servidor (sistema operativo de rede) é necessário planear a sua instalação - selecionar os componentes de hardware e software e perceber o que o servidor deve fazer. É crucial identificar as funções do servidor e os serviços de rede que o servidor precisará fornecer. E é também importante perceber quantas pessoas poderão vir a aceder ao servidor ao mesmo tempo.

Foi através da tecnologia de virtualização que criámos máquinas virtuais do Windows XP, Windows 8.1, Windows 10, Windows Server 2008 R2, Linux Lite e do Ubuntu Server. Esta tecnologia permite que vários sistemas operativos sejam executados simultaneamente numa única máquina.

Dentro das tarefas resolvidas ao longo da UFCD, aquela que me despertou mais interesse, ocorreu no Windows Server 2008 R2, quando foi instalado o Active Directory, promovemos o servidor a controlador de domínio, configurarmos o DNS Server e adicionámos um computador cliente ao domínio.

Esta UFCD exigiu muito trabalho autónomo, o que considero uma mais valia, para o desenvolvimento dos conhecimentos e das capacidades individuais.
