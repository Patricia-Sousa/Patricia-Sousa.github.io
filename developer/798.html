UFCD 798

Utilitários

Formador: Carlos Martins

Objetivos

Instalar e configurar utilitários sobre o sistema operativo (SO).

Reflexão

Um utilitário é um software desenvolvido para executar tarefas específicas. Este pode estar integrado no sistema operativo e ser um programa independente. Os recursos informáticos são geridos, mantidos e controlados por software utilitário. O software utilitário, que gere, mantém e controla os recursos do computador, está no centro da gestão do computador. Existem tarefas críticas, por exemplo, que um sistema não pode executar sem estas ferramentas.

Entre alguns destes programas (utilitários) estão os gestores de rede, o software antivírus, o software de proteção contra malware, as ferramentas de gestão de ficheiros, o software de backup e os compressores de ficheiros.

Nesta UFCD, como utilitários, também foram abordados os softwares de navegação na intranet/internet, os softwares de gestão de correio eletrónico, os softwares de gestão de comunicações e os gestores de discos.

Foram também vistos dois protocolos de rede – o FTP e o SSH. O FTP (File Transfer Protocol) é um protocolo para a transmissão de ficheiros entre computadores através de ligações TCP/IP. Numa transação FTP, o computador do utilizador final é normalmente designado por local host. O segundo computador envolvido no FTP é um remote host, que é normalmente um servidor. Ambos os computadores têm de estar ligados através de uma rede e configurados corretamente para transferir ficheiros por este protocolo. Os servidores devem ser configurados para executar este tipo de serviços e o cliente deve ter um software instalado, como o FileZilla (versão cliente), para aceder a estes serviços.

O SSH (Secure Shell Protocol ou Secure Socket Shell) é um protocolo de rede criptográfico para operar serviços de rede de forma segura através de uma rede não segura. Através do software PuTTY, conseguimos efetuar uma ligação SSH a um terminal de servidor remoto Linux.
