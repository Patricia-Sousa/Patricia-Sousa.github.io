UFCD 797

Sistemas Operativos - Tipologias

Formador: Carlos Martins

Objetivos

Instalar e configurar os sistemas operativos.

Reflexão

Nesta UFCD analisámos as exigências de um sistema operativo e criámos a máquina virtual(através do VirtualBox) para fazermos uma instalação limpa do Windows 10. Uma máquina virtual é uma tecnologia de virtualização que permite executar por exemplo, um sistema operativo separado da máquina real. Sendo necessária a imagem ISO para fazer a instalação, fizemos o download da mesma no site oficial da Microsoft.

Fizemos também personalizações e algumas configurações iniciais depois da instalação do sistema operativo e criámos um sistema de Logins para gerir diversos utilizadores. Aqui criámos um administrador e um utilizador padrão. Também configurámos algumas funcionalidades de segurança para a proteção do sistema, nomeadamente criar um ponto de restauro do sistema. Cada ponto de restauro, contém as informações necessárias para restaurar o sistema para o estado selecionado. Por último fizemos as atualizações necessárias no sistema operativo.

Considero que esta UFCD é importante, porque nos permite adquirir as bases necessárias para a instalação de um sistema operativo de raiz e possibilita-nos conhecer algumas personalizações e configurações, para uma utilização mais correta do mesmo.
