UFCD 800

Serviços Adicionais de Rede

Formador: António Fonseca

Objetivos

Instalar e configurar serviços adicionais de rede.

Reflexão

A UFCD, serviços adicionais de rede, tem como objetivo instalar e configurar serviços adicionais na rede, nomeadamente um file server e um print server ou um serviço de acesso remoto. Para tal, utilizámos o Windows Server 2019 Essentials e fizemos a sua instalação recorrendo ao programa de virtualização vmplayer. Nele fizemos algumas configurações, tais como - alterar o nome do servidor, atribuir um grupo de trabalho, alterar o idioma, atribuir um IPv4 estático ao servidor, habilitar o ambiente de trabalho remoto, instalação do AD, configuração do DNS, criação de uma unidade organizacional, criação de utilizadores e adicionar um computador cliente (Windows 10) ao domínio.

Instalámos também o Windows Admin Center no servidor. Esta ferramenta dá controlo total sobre todos os aspetos de uma infraestrutura de servidores e é útil para gerir servidores em redes privadas que não estão ligadas à internet.

Vimos também o que é um file server (servidor de ficheiros). Permite aos utilizadores a partilha de informações através de uma rede sem terem de transferir ficheiros fisicamente. Procedemos também à instalação do print server, como um role, a partir do server manager. Instalou-se 2 impressoras virtuais no servidor e foram partilhadas na rede e acedidas através do computador cliente.

Também foram alteradas permissões de um utilizador do domínio, de forma a que não seja possível aceder a uma determinada impressora. O software de gestão de impressão fornece aos administradores de sistemas/gestores de TI uma ferramenta centralizada para ativar, monitorizar e controlar a impressão numa organização.

Esta UFCD permitiu a aquisição de novos conhecimentos e sedimentou os conhecimentos da UFCD 799, sistemas de rede local, visto que ambas se relacionam.
