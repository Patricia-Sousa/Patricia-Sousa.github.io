UFCD 10871

Introdução à Administração de Sistemas

Formador: Tiago Mendes

Objetivos

Descrever os fundamentos das arquiteturas de redes.

Identificar os fundamentos de administração de sistemas operativos.

Realizar operações de administração de um sistema operativo.

Reflexão

Esta UFCD, na minha opinião, convergiu para a compreensão do papel do hardware, do software e dos procedimentos efetuados, para criar uma rede de comunicação entre computadores. Para este efeito, foi feito uma overview em arquitetura interna de computadores e em arquitetura de redes.

Ao nível do hardware, foram vistos todos os componentes, apontando para a necessidade de testar, reparar ou substituir fisicamente os mesmos, explicando também a necessidade do software para o seu funcionamento correto - dada a necessidade, também da sua instalação e configuração - passando pela evolução do conceito, a máquina, para o objeto que conhecemos hoje, como o computador. Um dispositivo complexo, para entrada e saída de dados, ou seja, informação.

A marca mais significativa, para mim, é a criação do transístor, um componente elétrico capaz de interromper sinais elétricos, intrínseco ao conceito de eletricidade. Dá assim, hipótese para o advento dos circuitos integrados, com a aplicação das portas lógicas e da álgebra de Boole.

Dada a dimensão dos componentes de um computador, a necessidade do seu correto funcionamento e das suas tarefas, o sistema operativo (software), é então o sistema capaz de gerir todos os recursos de hardware, de proporcionar uma interface gráfica para o utilizador, de executar diversos programas e ainda de detetar e responder a erros de software e hardware. Para complementar a aprendizagem, ainda foi visto passo a passo o seu processo de inicialização.

Esta UFCD, foi desenvolvida para mostrar a montagem e reparação de computadores, como uma saída profissional e mais uma skill curricular individual, pois qualquer bom profissional de TI deve estar apto para resolver problemas de hardware e software.

O conceito de rede de computadores nasce com o propósito de estabelecer comunicação e partilha de dados, visto que o computador, na atualidade é um veículo de trabalho entre as organizações.

Tendo em conta, tudo o que foi anteriormente mencionado, posso concluir que o hardware são todas as peças físicas e o software são todos os componentes lógicos construídos até hoje. Para este efeito, devemos ter em máxima consideração a segurança dos dados em comunicação e saber aplicar normas, protocolos, restrições, encriptações e modos de operação.

Foram dados a conhecer diversos softwares para simulações de montagem e organização de um computador, bem como softwares de benchmarking, para medir o desempenho dos seus componentes. Por último, desenvolvemos simulações de redes, no Cisco Packet Tracer, onde foram abordados muitos conceitos inerentes a uma infraestrutura de rede.
