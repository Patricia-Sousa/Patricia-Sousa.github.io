UFCD 3935

Programação em C#

Formador: Gisela Firmino

Objetivos

Produzir, depurar e testar o código de aplicações produzidas no ambiente Visual Studio com recurso à linguagem Visual C# .NET.

Manipular a programação por objetos e eventos em Visual C# .NET.

Utilizar o Visual C# .NET como interface para bases de dados.

Reflexão

"The hardest part of software development is naming things."

Esta UFCD constituiu-se na aprendizagem da linguagem de programação C#, para isso utilizámos o modo Console e o Visual C# .NET. O C# é uma linguagem de programação visual dirigida por eventos e totalmente orientada por objetos.

O modo console é usado para criar aplicações de linha de comando, que são executados no terminal ou prompt de comando. Para o Visual C#, utilizámos o Windows Forms, que é uma estrutura de interface de utilizador para a criação de aplicações para desktop do Windows.

A programação orientada para eventos e por objetos é mais complexa, no entanto, elas fazem sentido em ambiente gráfico.

Para contextualizar, um formulário, no Windows Forms, é um plano visual no qual são mostradas informações para o utilizador, como cliques ou utilização de teclas. Estas ações geram eventos que são descritos através do código.
