UFCD 10788

Fundamentos da Linguagem SQL

Formador: Rui Pereira

Objetivos

Identificar bases de dados relacionais.

Consultar dados armazenados dentro da base de dados.

Criar tabelas, inserir, alterar e apagar dados.

Reflexão

O SQL, Structured Query Language - Linguagem de Consulta Estruturada, é uma linguagem padronizada usada para gerir, consultar e manipular dados em base de dados relacionais.

O SQL permite aos programadores, criar, modificar e usar bases de dados de forma otimizada. Considero essencial estudar esta linguagem, pois esta, pode ser encontrada nas mais variadas aplicações, empresas e áreas. Durante as sessões desta UFCD, achei a linguagem SQL fácil de aprender e de se utilizar. Acho que ela é eficiente quando estamos a explorar dados relacionais e a cruzar tabelas.

Compreender esta linguagem pode ajudar a perceber melhor a estrutura de dados de uma empresa e a poupar tempo na localização de dados necessários. É uma skill fundamental para a era de desenvolvimento digital em que vivemos.
