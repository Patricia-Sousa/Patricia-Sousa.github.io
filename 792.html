UFCD 792

Criação de Páginas para WEB em Hipertexto

Formador: João Valente

Objetivos

Elaborar páginas para a web, com recurso a hipertexto.

Reflexão

Esta UFCD consistiu na criação de páginas para a web em HTML. HTML significa linguagem de marcação de hipertexto, sendo este a componente base da web. Isto quer dizer, que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, através do hipertexto. Os documentos de HTML podem ser interpretados pelos navegadores. Todos os documentos de HTML possuem marcadores, do inglês tags, que são palavras entre parênteses angulares. Esses marcadores são os comandos da formatação da linguagem, como pode ser visto na figura 1. Os conteúdos desta UFCD incidiram sobre: os conceitos gerais de HTML, as ligações, a formatação de texto com HTML, inserção de imagens, multimédia na web, animação na web, desenho de páginas web, inserção de tabelas, frames, e mapas. Para a avaliação desta UFCD, individualmente, fizemos a construção de um site de raiz.

O site deveria ser composto pelo menos com 5 páginas HTML, um parágrafo de texto com algumas formatações, um menu, uma tabela, uma imagem, um vídeo, um áudio, um iframe e uma imagem Map com uma área clicável.

Foi uma UFCD simples, onde não senti grandes dificuldades na componente prática. Consegui desenvolver o meu projeto de forma autónoma e concluí o mesmo com êxito.

Esta UFCD é uma componente importante no desenvolvimento de um programador web. É importante salientar que o HTML é o esqueleto de uma página web e que sem ele era impossível fazer a estrutura, definir o posicionamento dos elementos, estruturar a divisão de uma página em blocos visuais e combinar texto, imagens e vídeos. No entanto, tem as suas limitações, nomeadamente porque ele apenas permite criar uma versão estática de uma página web, o que o torna numa tecnologia limitada para a construção de páginas no padrão moderno.

Considero que adquiri os conteúdos desta UFCD e cumpri com os objetivos da mesma.
