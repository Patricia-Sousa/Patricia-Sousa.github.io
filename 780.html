UFCD 780

Aplicações de Gestão Administrativa

Formador: João Valente

Objetivos

Definir o plano de instalações, de requisitos funcionais, de acessos e segurança de aplicações informáticas de gestão administrativa.

Instalar, parametrizar e configurar aplicações informáticas de gestão administrativa.

Reflexão

A informação é de vital importância para qualquer organização. As atividades de uma organização são traduzidas em valores numéricos, que permitirão aos gestores tomar decisões mais corretas para uma determinada situação e num determinado momento. As organizações precisam de armazenar as suas informações de uma forma estruturada. Desta forma, um software de gestão é essencial para uma boa gestão de uma empresa. Antes de mais, gostaria de definir o que é a gestão e a administração. A gestão é a atividade que tem como princípio fundamental incentivar a participação, estimular a autonomia e a responsabilidade dos funcionários. A administração significa planear algo, controlar e dirigir os recursos humanos, materiais e financeiros. Através deste raciocínio, posso definir a gestão administrativa como uma solução que visa encontrar uma forma racional de projetar uma organização de um negócio como um todo.

Esta UFCD teve especial atenção nos softwares de gestão - o projeto Colibri e o Jasmin. O software de gestão é uma ferramenta essencial para o crescimento sustentado e para uma melhor administração de empresas.

O projeto Colibri é um software de gestão comercial multiplaforma certificado que aborda as principais necessidades da área comercial de uma pequena e média empresa. Este software tem várias aplicações integradas como: a gestão de stocks, a criação e organização de clientes e fornecedores, a configuração de email automático, o controlo de vendas e emissão de documentos. Entre outros atributos, ele permite armazenar todas as informações de clientes e com a formulação de relatórios consegue-se ter uma visão geral de todos os dados.

O Jasmin é um software de gestão na cloud (o termo geralmente é usado para descrever centros de dados disponíveis para muitos utilizadores pela internet), é muito simples de usar e basta apenas ter acesso à internet e aceder a um browser para ter fácil acesso.

Para além destas ferramentas de apoio à gestão, abordámos o programa, o Excel, que é uma folha de cálculo com comandos avançados para a análise e visualização de dados.

Através desta UFCD posso concluir que a gestão administrativa envolve utilizações nas diversas áreas de conhecimento, como a contabilidade, a estatística, a economia, a psicologia, o direito, a sociologia e a informática. E para uma boa gestão empresarial, uma empresa deve incluir um software de gestão porque agiliza e automatiza os processos, melhora a tomada de decisões, simplifica a gestão, reduz os custos, entra em conformidade com a legislação, traz segurança na informação e ajuda no controlo de prazos. O maior objetivo da gestão administrativa é aumentar a produtividade e lucratividade da empresa, até porque como nós vimos durante esta UFCD, uma gestão eficiente vai muito além de uma rotina de trabalho e um ambiente saudável e para isso, deve-se planear todas as etapas, comunicar de maneira eficiente e usar a tecnologia a nosso favor.
